JSON-to-MoonBit

Convert JSON to MoonBit code

This tool instantly converts JSON into MoonBit code. Paste a JSON structure on the left and the equivalent MoonBit code will be generated to the right, which you can paste into your program. The script has to make some assumptions, so double-check the output!

For an example, try converting JSON from the SmartyStreets API or the GitHub API.
